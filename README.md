# ARTEN — webbplats

Statisk webbplats för ARTEN (marknadsföring och juridisk information).

Publiceras via GitHub Pages: https://zeffou.github.io/arten-web/

## Sidor

- `index.html` — startsida
- `integritet.html` — integritetspolicy
- `villkor.html` — användarvillkor
- `support.html` — support och FAQ
- `kallor.html` — obligatoriska attributioner

## Lokalt

Öppna valfri HTML-fil i webbläsaren, eller kör en enkel server:

```bash
python3 -m http.server 8080
```

## Placeholders

Uppdatera `kontakt@arten.app` när du har e-post kopplad till domänen.
